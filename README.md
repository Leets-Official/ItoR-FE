## ✍🏻 프로젝트 개요

MoodMate 무드메이트는 매일 밤 10시에 데이트 무드 취향, 나이와 학과 등을 고려하여 1대1 매칭을 해주는 서비스 입니다.
1:1 채팅은 매칭 시작 후 22시간, 다음날 오후 8시까지 유지됩니다.
